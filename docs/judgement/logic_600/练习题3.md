  1. 蛇纹石是两个大陆板块在运动中相互碰撞时挤压海底大陆而形成的一种岩石

  3. 蛇纹石中碳的形状呈现出生物组织特有的管状和洋葱型结构，类似于早期的海洋微生物
  4. 由于大陆板块的运动才创造出了许多新的大陆，在板块运动发生之前，地球上绝大部分地区是一片汪洋大海

  3. 采访对象只有用含混不清或枯燥无味的话来应付，才能回答那些空泛、笼统的提问
